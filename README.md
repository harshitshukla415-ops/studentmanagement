# Student Management System

A Python-based web application for managing student information and records efficiently.

## Overview

This Student Management System is a user-friendly application built with Streamlit that provides comprehensive tools for managing student data, records, and related operations.

## Features

- 📚 **Student Database Management** - Add, update, and manage student records
- 👥 **Student Information Tracking** - Keep track of student details and progress
- 📊 **Data Organization** - Efficiently organize and retrieve student information
- 🎯 **User-Friendly Interface** - Simple and intuitive web-based interface

## Tech Stack

- **Language**: Python
- **Framework**: Streamlit
- **Platform**: Web-based application

## Requirements

- Python 3.7+
- Streamlit

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/harshitshukla415-ops/studentmanagement.git
cd studentmanagement
```

### 2. Set Up Virtual Environment (Optional but Recommended)

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run main.py
```

The application will start on `http://localhost:8501` by default.
